Résumé des commandes Docker

Commandes de base

Commande Description Exemple
docker run Lance un conteneur à partir d’une image docker run -it ubuntu /bin/bash
Crée et démarre un conteneur interactif avec Ubuntu
docker ps Liste les conteneurs en cours d’exécution docker ps
Affiche tous les conteneurs actifs
docker ps -a Liste tous les conteneurs (actifs et stoppés) docker ps -a
docker stop Arrête un conteneur en cours d’exécution docker stop CONTAINER_ID
docker rm Supprime un conteneur stoppé docker rm CONTAINER_ID

Gestion des images

Commande Description Exemple
docker images Liste les images disponibles localement docker images
docker pull Télécharge une image depuis Docker Hub (ou un registry privé) docker pull ubuntu
docker rmi Supprime une image locale docker rmi IMAGE_ID
docker build Construit une image à partir d’un Dockerfile docker build -t mon-image .

Logs et monitoring

Commande Description Exemple
docker logs Affiche les logs d’un conteneur docker logs CONTAINER_ID
--follow pour suivre en temps réel
docker top Montre les processus tournant dans un conteneur docker top CONTAINER_ID
docker stats Monitore en temps réel l’utilisation des ressources par les conteneurs docker stats

Exécution et gestion avancée

Commande Description Exemple
docker exec Exécute une commande à l’intérieur d’un conteneur en cours d’exécution docker exec -it CONTAINER_ID /bin/bash
docker start Redémarre un conteneur précédemment stoppé docker start CONTAINER_ID
docker network Gère les réseaux Docker (création, suppression, inspection) docker network create mon-reseau
Crée un nouveau réseau

Exemple de Dockerfile

Voici un exemple rapide d’un Dockerfile pour une image Node.js :

FROM node:14-alpine
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
EXPOSE 3000
CMD ["npm", "start"]
